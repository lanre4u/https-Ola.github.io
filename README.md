 VR Golf Simulation

A VR golf training simulation built with Unity and XR Interaction Toolkit.  
Supports Meta Quest headset and includes teleportation and physics-based ball interaction.

 Features
- Teleportation and player movement
- Physics-based golf ball mechanics
- Score tracking system

 Tools Used
- Unity 2021.3
- C#
- XR Interaction Toolkit

How to Run
1. Clone the repository
2. Open in Unity 2021.3
3. Press Play to test
